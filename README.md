BombPay
=============

간편 분할 결제 서비스
-------------

## 개요
 2016 금융API를 활용한 핀테크 해커톤에 출전하여 개발한 "BoomPay"라는 애플리케이션입니다. 방장이 방을 만들면 QR코드를 생성되고, 사용자들이 QR코드를 인식하여 간편한 결제를 할 수 있습니다. 실제로 해당 앱은 가상계좌를 발급받아 출/입금이 가능하며 금융API 사용을 위해서 'KISA'에서 제공하는 네트워크를 이용해야합니다.
